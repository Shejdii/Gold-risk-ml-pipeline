# Gold Risk ML Pipeline

End-to-end MLOps pipeline for ingesting gold market data (XAUUSD),
engineering risk features, and training reproducible ML models.

---

## Project Goal

This project demonstrates production-oriented MLOps practices:

- Reproducible data ingestion
- Structured feature engineering
- Modular training pipeline
- Code quality automation (Makefile, linting, testing)
- Containerization (planned)
- CI/CD integration (planned)

---

## Tech Stack

- Python 3.11+
- Pandas
- Requests
- Pytest
- Pylint
- Black
- Make
